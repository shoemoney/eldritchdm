  `group_by: ["archetype"]`.
- `time_window: "30d"`.
- `query_recipe`: instructs how to load JSON-lines run artifacts, plot
  overall_score over time (line), per_dimension_mean breakdown (bar),
  per_archetype_mean scoreboard (table).

**Done when:** valid JSON; OUR-FORMAT v1 conformant; the `query_recipe`
explicitly flags the offline-source caveat so future readers don't expect
