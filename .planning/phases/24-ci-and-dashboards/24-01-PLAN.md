# Plan 24-01 — GitHub Actions CI matrix (POLISH-01)

## Objective

Add cross-platform CI at `.github/workflows/ci.yml` covering macOS + Linux on Python 3.11.
Verifies Phase 14 FLAKE-01 skip-gates work cleanly when `[mac-ocr]` and `[observability]`
extras are NOT installed (the default off-path). A separate optional `extras-mac` job
exercises the full-stack on-path.

## Context

@.planning/phases/24-ci-and-dashboards/24-CONTEXT.md (D-186, D-187)
@pyproject.toml (optional-dependencies: mac-ocr, linux-ocr, observability, dev)

## Tasks

### Task 1 (auto) — Create `.github/workflows/ci.yml`

**Files:**
- `.github/workflows/ci.yml` (new)

**Implementation:**
- Trigger: `push` to any branch + `pull_request` to main.
- Job `test` with matrix `os: [macos-latest, ubuntu-latest]`, `python: ["3.11"]`.
- Steps per matrix cell:
  1. `actions/checkout@v4`
  2. `astral-sh/setup-uv@v3` (latest stable)
  3. `actions/setup-python@v5` with `python-version: ${{ matrix.python }}`
  4. `uv sync --extra dev` (NO `--extra mac-ocr`, NO `--extra observability`)
  5. `uv run ruff check src/ tests/ run.py`
  6. `uv run lint-imports`
  7. `uv run pytest tests/ -q`
- Linux-only extra steps (gated on `runs-on == 'ubuntu-latest'`):
  8. `bash scripts/ci/check_safe_yaml.sh`
  9. `bash scripts/ci/check_summary_frontmatter.sh`
- Separate job `extras-mac`:
  - `runs-on: macos-latest`, `continue-on-error: true`
  - `uv sync --extra dev --extra mac-ocr --extra observability`
  - `uv run pytest tests/ -q`
  - Marked informational (not a required check).
- Concurrency group keyed on `${{ github.workflow }}-${{ github.ref }}` with
  `cancel-in-progress: true` so stacked pushes don't pile up.

**Verification:**
```bash
# Local YAML sanity (no python yaml unsafe-load; just structural):
python -c "import yaml; yaml.safe_load(open('.github/workflows/ci.yml'))"
# Ruff + lint-imports + pytest must all still pass locally:
uv run ruff check src/ tests/ run.py
uv run lint-imports
uv run pytest tests/ -q
```

**Done when:**
- `.github/workflows/ci.yml` exists and parses as valid YAML
- Default matrix does NOT install `[mac-ocr]` or `[observability]`
- Optional `extras-mac` job exists with `continue-on-error: true`
- Linux runner runs `check_safe_yaml.sh` + `check_summary_frontmatter.sh`
- Local ruff + lint-imports + pytest all pass

## Success Criteria

1. Workflow YAML structurally valid and parses with `yaml.safe_load`
2. Default matrix installs only `[dev]` extras
3. `extras-mac` job is non-gating (`continue-on-error: true`)
4. Linux-only CI scripts present in workflow
5. No regression in local ruff / lint-imports / pytest
