# Plan 23-02 — AOE addendum live prompt assembly with versioning

## Objective

Tighten the SmartMonsterDriver's system-prompt assembly so the AOE addendum is
injected ONLY when the current actor has ≥2 AOE-class actions. When the addendum
IS appended, stamp the OTel decision-span with its SemVer.

This finishes Phase 20's "ship-but-not-wired" gap (WIRE-03 / D-180 / D-181 / D-182).

## Context

- The AOE addendum text + version are already loaded on
  `SmartMonsterDriver.__init__` (`smart_monster_driver.py:349-366`) into
  `self._aoe_addendum_text` and `self._aoe_addendum_version`.
- Current predicate at line 576 — `if action_descriptors and self._aoe_addendum_text:` —
  fires whenever the actor has ANY validated action, regardless of kind. That's
  too loose; a monster with only `single` and `multi_attack` actions does not
  benefit from the multi-target tactic guidance.
- `ActionDescriptor.kind` is `Literal["single", "aoe", "multi_attack", "breath", "cone"]`
  (line 111). D-180 defines AOE-class as `{"aoe", "cone", "breath"}`.
- The decision span is created in `_choose_target` at line 436 and threaded into
  `_pick_target_llm` as the `span` kwarg (line 478). Stamping the version onto the
  same span is the correct OBS-tie-in (no nested spans, per D-66).

def get_addendum_version(path: Path | None = None) -> str:
    """Return ONLY the SemVer string from the AOE addendum file.

    Convenience wrapper for callers that need the version without the body
    (e.g., for OTel span attributes when the body is already cached).

    Raises:
        AoeAddendumError: same conditions as ``load_aoe_addendum``.
    """
    _text, version = load_aoe_addendum(path)
    return version
```

# D-180: addendum injection requires ≥2 AOE-class actions on the actor.
# D-181: when injected, stamp the decision-span with the addendum SemVer.
aoe_count = sum(
    1 for a in action_descriptors if a.get("kind") in {"aoe", "cone", "breath"}
)
if aoe_count >= 2 and self._aoe_addendum_text:
    system_prompt = legacy_system_prompt + "\n\n" + self._aoe_addendum_text
    if span is not None:
        span.set_attribute(
            "eldritch.aoe.addendum_version", self._aoe_addendum_version
        )
else:
    system_prompt = legacy_system_prompt
```
