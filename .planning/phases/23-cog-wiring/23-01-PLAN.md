# Plan 23-01 — `/end_game` slash command + MonsterMemoryRegistry exposure

## Objective

Wire the session-close path so that ending a campaign in Discord:

1. Closes the dm20 claudmaster session.
2. Purges per-session monster memory (cumulative damage / concentration /
   marked-dangerous signals) from the in-process registry.
3. Returns the channel row to `ChannelState.LOBBY`.
4. Surfaces an ephemeral confirmation embed to the invoker.

## Context

- `MonsterMemoryRegistry.purge_session(channel_id, session_id) -> int` already exists
  and is fail-soft (`src/eldritch_dm/gameplay/monster_memory.py:307`).
- `bot.py` constructs the smart monster driver at `setup_hook` (line 412) but does
  NOT currently construct or thread a `MonsterMemoryRegistry`. Without registry
  exposure on `bot`, the new `/end_game` purge would target a separate (and empty)
  instance. **Registry exposure is a prerequisite.**
- D-178 / D-179 govern the session-close ordering and the fail-soft contract.
- The concentration half of v1.6 WIRE-02 (originally inferred from D-177) has been
  moved into the deferred WIRE-01 block in REQUIREMENTS.md per 23-HALT-REPORT.md.

### Task 2 — Add `/end_game` slash command to `LobbyCog`

`type=auto`

**Files:**
- `src/eldritch_dm/bot/cogs/lobby.py`

**Implementation:** Add a new `@app_commands.command(name="end_game", ...)` method on
`LobbyCog`. The handler runs the D-178 sequence:

1. **Defer first (EDM001 — D-09):** `await interaction.response.defer(thinking=True, ephemeral=True)`.
2. **Permission gate (T-03-03 parity with `/load_adventure`):**
   `can_act_on_character(interaction, character_player_id=None)` — DM-only.
3. **Fetch active session:**
   `session = await self.bot.channel_sessions.get(str(interaction.channel_id))`.
   If `None`, ephemeral "No active campaign in this channel — nothing to end."
4. **End dm20 claudmaster session (best-effort, fail-soft per D-179):**
   ```python
   if session.claudmaster_session_id:
       try:
           await self.bot.mcp.call(
               "dm20__end_claudmaster_session",
               session_id=session.claudmaster_session_id,
           )
       except Exception:
           bound_log.exception("end_game_dm20_close_failed")
           # NON-FATAL: continue to purge memory + LOBBY-flip regardless.
   ```
5. **Purge monster memory (fail-soft — `purge_session` already swallows exceptions
   and returns 0 on failure):**
   ```python
   registry = getattr(self.bot, "monster_memory_registry", None)
   purged = 0
   if registry is not None and session.claudmaster_session_id:
       try:
           purged = registry.purge_session(
               str(interaction.channel_id),
               session.claudmaster_session_id,
           )
       except Exception:
           bound_log.exception("end_game_memory_purge_failed")
   ```
6. **Flip state to LOBBY (preserves the channel row as audit; matches halt-report §"WIRE-02 UNBLOCKED"):**
   ```python
   await self.bot.channel_sessions.upsert(
       channel_id=str(interaction.channel_id),
       campaign_name=session.campaign_name,
       claudmaster_session_id=None,           # session is gone
       dm20_party_token=session.dm20_party_token,
       state=ChannelState.LOBBY,
   )
   ```
7. **Ephemeral confirmation embed:**
   ```python
   embed = discord.Embed(
       title="🛑 Session Ended",
       description=(
           f"Campaign **{session.campaign_name}** has been closed.\n\n"
           f"Monster memory cleared ({purged} entries).\n"
           f"Channel returned to LOBBY."
       ),
       color=0xED4245,
   )
   embed.set_footer(text="🎲 ShoeGPT · EldritchDM")
   await interaction.followup.send(embed=embed, ephemeral=True)
   ```
