    ```
    # aoe-addendum-version: 1.0.0
    EXTENSION: multi-target tactic selection.

    You may now select MORE THAN ONE target when the monster has an
    `available_actions` entry whose `kind` is `aoe`, `breath`, `cone`, or
    `multi_attack`. Emit JSON of the form:

      {
        "target_pc_ids": ["<id1>", "<id2>", ...],
        "tactic_kind": "<single|aoe|multi_attack|breath|cone>",
        "rationale": "<one short sentence>"
      }

    Selection heuristic:
      - `single`: prefer when only one PC is within the chosen action's
        `range_ft`, or when the monster has no AOE action available.
      - `aoe` / `breath` / `cone`: prefer when 2+ PCs are within the
        action's `range_ft`. Choose the subset that maximises overlap.
      - `multi_attack`: a single-target action that lands more than one
        attack on the SAME PC; emit one id in the list.

    Constraints:
      - Every id in `target_pc_ids` MUST appear in the candidate list.
      - `single` → exactly 1 id. `aoe`/`breath`/`cone` → 2+ ids.
        `multi_attack` → 1+ ids.
      - Do NOT compute exact HP, AC, or class. Refer to visible cues only
        (armor, posture, visible wounds, active conditions).
    ```

    **`prompts/aoe_addendum.py`** — loader (mirrors `eval/judge_prompt.py`):

    ```python
    """AOE-addendum prompt loader with SemVer header (Phase 20 / D-152)."""

    from __future__ import annotations

    import re
    from pathlib import Path

    _VERSION_HEADER_RE = re.compile(r"^# aoe-addendum-version: (\d+\.\d+\.\d+)\s*$")
    _DEFAULT_PROMPT_PATH = Path(__file__).parent / "aoe_addendum.txt"


    class AoeAddendumError(Exception):
        """Raised when the AOE addendum file is missing or malformed."""


    def load_aoe_addendum(path: Path | None = None) -> tuple[str, str]:
        target = path if path is not None else _DEFAULT_PROMPT_PATH
        if not target.is_file():
            raise AoeAddendumError(f"AOE addendum not found: {target}")
        text = target.read_text(encoding="utf-8")
        if not text:
            raise AoeAddendumError(f"AOE addendum is empty: {target}")
        first_line = text.splitlines()[0]
        match = _VERSION_HEADER_RE.match(first_line)
        if match is None:
            raise AoeAddendumError(
                f"AOE addendum {target} line 1 must match "
                f"'# aoe-addendum-version: X.Y.Z'; got: {first_line!r}"
            )
        return text, match.group(1)
    ```
