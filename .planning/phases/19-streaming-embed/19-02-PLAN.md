# Phase 19 Plan 02 ŌĆö Hidden fallback + STREAM_ENABLED + cog wiring (STREAM-02, STREAM-03)

## Objective

1. Add `STREAM_ENABLED` Settings field (default `True`, env-gated). When false,
   suppress the thinking indicator entirely (STREAM-03).
2. Wire `MonsterDriverFactory` to forward an optional
   `embed_update_callback` kwarg through to `SmartMonsterDriver`. The bot
   constructs a callback closure that dispatches per-channel through
   `CombatCog._coalescers` (STREAM-01 wiring complete).
3. Guarantee fallback path stays hidden ŌĆö verify in tests that the LLM-
   timeout / refusal / hallucinated-id paths produce NO additional embed
   update with "fallback"/"AI failed" wording (STREAM-02). Only the single
   "­¤żö ... sizing up" line ever appears on the embed via this code path; the
   resolved-action embed flows through CombatCog's existing post-turn update.

## Context

- Settings is pydantic-settings v2 frozen=True; new field needs an alias
  `STREAM_ENABLED` consistent with `MCPCACHE_ENABLED`/`MONSTER_DRIVER`.
- Driver factory must accept `embed_update_callback` and:
  - When mode = `random`: pop it from kwargs (random driver doesn't take it).
  - When mode Ōłł {`smart`, `mixed`}: pass it through to `SmartMonsterDriver`.
- Cog wiring: the `bot.py` factory call must:
  - Build a closure `async def _emit_thinking(channel_id, text)` that looks
    up the channel's combat coalescer on `CombatCog._coalescers`.
  - Honor `settings.stream_enabled` ŌĆö if false, pass `embed_update_callback=None`.
- The closure builds a fresh embed with `description = text` plus the COMBAT
  color, sends via the coalescer's existing `update(embed, view=...)` API.
  Wrapped in `contextlib.suppress(Exception)` (D-145).

```python
async def _emit_thinking_indicator(channel_id: str, text: str) -> None:
    """Bot-level closure that routes thinking-indicator text through the
    per-channel CombatCog coalescer. Cancellation-safe (D-145).
    """
    combat_cog = self.get_cog("CombatCog")
    if combat_cog is None:
        return
    coalescer = getattr(combat_cog, "_coalescers", {}).get(channel_id)
    if coalescer is None:
        return
    embed = discord.Embed(
        title="ŌÜö’ĖÅ Combat",
        description=text,
        color=int(EmbedColor.COMBAT),
    )
    with contextlib.suppress(Exception):
        await coalescer.update(embed)

stream_cb = _emit_thinking_indicator if settings.stream_enabled else None
```

Pass `embed_update_callback=stream_cb` to `make_monster_driver(...)`.
