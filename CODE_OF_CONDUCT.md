# Code of Conduct

EldritchDM is an open-source hobby project. We want it to stay a friendly,
welcoming place for tabletop fans and contributors of every experience level.

## Our Standards

**Be kind. Be patient. Assume good faith.** Tabletop is a hobby, and so is
this project — please keep it a friendly place to be.

- Harassment, personal attacks, slurs, and bad-faith argument are not welcome.
- Disagreement on technical direction is fine; disagreement that targets a
  person is not.
- Constructive criticism is welcome. Phrase it about the work, not the person.
- Newcomers ask "obvious" questions. We were all newcomers once. Be helpful.

## Scope

This Code of Conduct applies within all project spaces — the GitHub
repository, issue tracker, pull requests, discussions, and any community
spaces where someone is representing the project.

## Enforcement

If something feels off — an interaction in an issue, a PR comment, a
discussion thread — email **jeremy@shoemoney.com** directly. Don't escalate
it in the public issue tracker.

The maintainer will:
1. Acknowledge your report within 48 hours
2. Investigate privately and respectfully
3. Take whatever action is needed: warning, edit, lock, ban
4. Follow up so you know it was addressed

Maintainer decisions on enforcement are final. The point is to keep this
project welcoming, not to litigate every disagreement.

## Attribution

This Code of Conduct is project-specific and intentionally lightweight. It
is not a formal adoption of the Contributor Covenant — we want clarity over
ceremony. If you prefer a more detailed framework, see the
[Contributor Covenant](https://www.contributor-covenant.org/).

For contribution guidelines (technical scope, PR process, commit style), see
[CONTRIBUTING.md](./CONTRIBUTING.md).
